Overview of library preparation and sequencing:
  1. Start from 400 µl frozen serum sample.
  2. Extract total RNA with double elution for maximal template recovery.
  3. cDNA synthesis and 30 cycles of PCR using life technologies SuperScript® III One-Step RT-PCR System with Platinum® Taq High Fidelity DNA Polymerase. PCR is done separately for six overlapping fragments labeled F1 through F6.
  4. Library preparation using Illumina Nextera XT® sample preparation kit, followed by column purification. Size selection with the Sage Science BluePippin® with cutoffs at 550 and 900 bp. These sizes include the library adaptors.
  5. Sequencing on illumina MiSeq® using 500 or 600 cycles kit.
  6. Mapping and cleaning of the sequencing reads (in house Python application).
