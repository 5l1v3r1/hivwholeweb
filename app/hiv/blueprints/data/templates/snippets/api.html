All data shown on this website can be accessed via a RESTful API for easy web-compatible manipulation.

The API is in JSON format only. Any "Accept" HTTP header will be ignored.

The base URI is /api/data/. To this root you can add parameters of interest:

  • TREES: tree/<patientname>/<region> (e.g. tree/p1/V3)
  • VIRAL LOAD: viralLoad/<patientname> (e.g. viralLoad/p1)
  • CD4+ CELL COUNT: cellCount/<patientname> (e.g. cellCount/p1)
  • NUMBER OF TEMPLATE MOLECULES: numberTemplates/<patientname> (e.g. numberTemplates/p1)
  • REFERENCE SEQUENCES: referenceSequence/<patientname> (e.g. referenceSequence/p1)
  • DIVERGENCE/DIVERSITY: divdiv/<patientname>/<region> (e.g. divdiv/p1/V3)
  • COVERAGE: coverage/<patientname> (e.g. coverage/p1)
  • SINGLE NUCLEOTIDE POLYMORPHISMS: singleNucleotidePolymorphisms/<patientname> (e.g. singleNucleotidePolymorphisms/p1)
  • HAPLOTYPES: haplotypes/<patientname>/<region> (e.g. haplotypes/p1/V3)
